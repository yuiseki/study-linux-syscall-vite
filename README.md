# Linux Syscall Quiz

日本人ユーザーが「それっぽい英単語」を見て、どれがLinuxシステムコールかを当てるクイズゲームです。

## 特徴

- 🎯 三択形式のクイズゲーム
- 🎉 派手な演出効果（コンフェッティ、パーティクル、画面シェイク）
- 🔊 サウンドエフェクト（Web Audio API）
- ⚙️ カスタマイズ可能な設定（難易度、出題数、演出強度）
- 📊 詳細な成績管理と統計
- ⌨️ キーボード操作対応
- ♿ アクセシビリティ対応
- 📦 完全オフライン動作

## セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build

# プレビュー
npm run preview
```

## 技術スタック

- ⚛️ React 18
- ⚡ Vite 5
- 📘 TypeScript
- 🎨 CSS-in-JS（インラインスタイル）

## 要件

詳細な要件については `.kiro/specs/linux-syscall-quiz/requirements.md` を参照してください。

## 操作方法

### キーボードショートカット

- `1`, `2`, `3`: 選択肢を選ぶ
- `Enter`: 次の問題へ進む

### マウス/タッチ

- 選択肢をクリック/タップして回答
